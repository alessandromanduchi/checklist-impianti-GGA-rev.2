### 6. PWA Install

## Come Catturare Screenshots

### Desktop (Chrome)
1. Apri l'applicazione in Chrome
2. Premi F12 per aprire DevTools
3. Clicca device toolbar (Ctrl+Shift+M)
4. Seleziona dimensione desiderata
5. Premi Ctrl+Shift+P
6. Cerca "Screenshot"
7. Seleziona "Capture full size screenshot"

### Mobile
Cattura screenshot direttamente dal dispositivo
